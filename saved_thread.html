Lifter ticking is pretty normal on these engines, but if it really bothers you, Ive got a solution. If youre still using the factory BMW 0W-30 oil, try switching to Liqui Moly 5W-30 Special Tec B FE.
Ive got a 2017 M2 with 70K miles. I had always used the OEM BMW oil, but just recently did my first switch  and the difference is night and day. That annoying lifter tick? Completely gone. Car sounds and feels smoother. Definitely worth a try if youre dealing with the same issue.
